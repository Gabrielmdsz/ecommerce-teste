# E-commerce de Tênis

Este é um projeto de e-commerce simples para uma loja de tênis e calçados modernos. O site apresenta uma interface responsiva com seções para navegação, destaques de produtos, categorias e inscrição em newsletter.

## Descrição

O projeto consiste em uma página web estática que simula uma loja online de tênis. Inclui um cabeçalho com navegação, uma seção hero com banner, categorias de produtos, uma galeria de produtos em grid e um rodapé com links e redes sociais.

## Funcionalidades

- **Cabeçalho responsivo**: Navegação principal com menu mobile.
- **Seção Hero**: Destaque com imagem de fundo e call-to-action.
- **Categorias**: Carrossel de categorias de tênis (Casual, Esporte, Moderno, Futurista).
- **Galeria de Produtos**: Grid de imagens de produtos com layout bento.
- **Newsletter**: Formulário para inscrição por e-mail.
- **Rodapé**: Links organizados e ícones de redes sociais.

## Tecnologias Utilizadas

- **HTML5**: Estrutura da página.
- **CSS3**: Estilização modular com arquivos separados para componentes.
- **Font Awesome**: Ícones para redes sociais e navegação.

## Como Usar

1. Clone ou baixe o repositório.
2. Abra o arquivo `index.html` em um navegador web.
3. Navegue pelas seções do site.

Não há necessidade de servidor ou instalação adicional, pois é uma página estática.

## Responsividade

O site é responsivo e se adapta a dispositivos móveis e desktop, com imagens otimizadas via `<picture>` e media queries.
